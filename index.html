Projects

Megan E. McCarty Collection

A personal project showcasing insect specimens in a collection. Includes site-wide Google Custom Search; a specimen record table that can be searched, sorted, and filtered; individual pages for each specimen and its associated data; Google Maps pinpointing site of collection for every specimen with GPS data; responsive design; and beautiful image galleries. Built using Django.

Studio413

An art blog built with Django and Wagtail CMS, with features including draft posts, comments, a built-in newsletter backend using Sendgrid, email notifications for new comments and subscribers, and user registration. Integrated with Etsy's API to showcase art pieces, as well as how many views and favorites they have received. Site is fully responsive and displays beautifully on mobile devices.

Dr. David Adamski

A website displaying scientific accomplishments. Features include responsive design and easy ability to add items (like publications, presentations, and travels) to a database, with the corresponding pages updating dynamically. Includes a simple search ability for easily finding a specific item in a list. Site displays beautifully on mobile devices. Built with Django.

Frumenta Partners, LLC

A professional website for a business consulting firm built with Django and Wagtail CMS. The site includes a parallax effect on the home page, a contact form, a news section (akin to a blog), and the ability for the owner to easily edit the site's content through the Wagtail CMS. The site displays beautifully on mobile devices and is fully responsive.
